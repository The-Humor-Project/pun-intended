## Repository instructions

- Fonts: use only web-safe fonts; do not load fonts from external services or local font files.
- Allowed stacks: `Georgia, "Times New Roman", Times, serif`, `"Trebuchet MS", Verdana, Tahoma, sans-serif`, `"Courier New", Courier, monospace`, `"Lucida Sans", "Lucida Sans Unicode", "Lucida Grande", sans-serif`.

## Supabase schema
- The latest database types live in src/types/supabase.ts
- This is generated by running npm run supabase:types
- The user does not want to use a SUPABASE_ROLE_KEY.  All actions should use the logged-in user's JWT key.

## Design
- The non-admin area of the site should be very round and curvy
- The admin area of the site should be very sharp, with no round corners
